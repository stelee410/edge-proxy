# Web Search（网络搜索）

根据用户需求进行网络搜索，抓取结果页的详细内容并整理成文字报告返回。

- **阶段**: mid_conversation  
- **类型**: code (handler: web_search)  
- **参数**:
  - `query`（必填）— 搜索关键词或用户的具体信息需求  
  - `max_results`（可选）— 最多抓取几条结果，1–10，默认 5  
  - `max_chars_per_page`（可选）— 每页最多保留字符数，默认 6000  

**流程**：通过 DuckDuckGo HTML 搜索获取结果链接 → 逐个请求页面 → 提取正文并转为可读文本 → 按「标题 + 链接 + 内容摘要」整理成一份报告返回给用户。
